Privacy Policy for YouTube Bookmark Manager

Last updated: February 15, 2025

Permissions and Their Usage

This extension requires the following permissions:

  • YouTube.com Host Permission: Required to add bookmark functionality to YouTube's video player, read video titles, and access timestamps. The extension only operates on YouTube.com and does not interact with any other websites.
  • Storage Permission: Used to save your bookmarks locally on your device. No data is sent to external servers.
  • Tabs Permission: Used only to open YouTube videos when you click on a bookmark. The extension cannot access content from other websites.

Data Collection and Usage

The YouTube Bookmark Manager extension:

  • Does not collect, transmit, or store any personal information
  • Does not track your browsing history
  • Does not use cookies
  • Does not share any data with third parties
  • All bookmark data is stored locally on your device using Chrome's storage API

Data Storage

The following information is stored locally on your device:

  • YouTube video IDs of bookmarked videos
  • Video titles
  • Timestamp information
  • User-created tags and color selections

Third-Party Services

This extension:

  • Interacts only with YouTube's public interface
  • Does not send data to any third-party services
  • Does not use any external APIs except YouTube's public interface
  • Does not collect analytics or usage data

Data Removal

All stored data can be removed by:

  • Using the delete functions within the extension
  • Uninstalling the extension
  • Clearing browser storage for the extension

Updates

This privacy policy may be updated to reflect changes in the extension's functionality or requirements. Users will be notified of any significant changes through the Chrome Web Store update system.
