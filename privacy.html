Privacy Policy for YouTube Bookmark Manager

Last updated: February 15, 2025

Data Collection

The YouTube Bookmark Manager extension does not collect, transmit, or store any personal information. All bookmark data is stored locally on your device using Chrome's storage API.

Data Storage

The following information is stored locally on your device:

  • YouTube video IDs of bookmarked videos
  • Video titles
  • Timestamp information
  • User-created tags and color selections

Third-Party Services

This extension interacts only with YouTube's public interface and does not send data to any third-party services.

Data Removal

All stored data can be removed by:

  • Using the delete functions within the extension
  • Uninstalling the extension
